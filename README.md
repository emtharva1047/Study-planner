# Study-planner
📚 Study Planner + Pomodoro Timer
🚀 Overview

Study Planner + Pomodoro Timer is a productivity web application that helps users manage their study schedule efficiently using task tracking and the Pomodoro technique (focused study intervals with breaks).

✨ Features
Add, edit, and delete study tasks
Set task priorities (High / Medium / Low)
Pomodoro timer (25 min focus + 5 min break)
Start / Pause / Reset timer
Track completed sessions
Daily progress tracking
Motivational notifications
Dark mode toggle
🛠️ Tech Stack
HTML5
CSS3
Bootstrap 5
JavaScript
jQuery
AJAX
localStorage
📊 Key Functionalities
Real-time countdown timer
Persistent data using localStorage
Dynamic DOM updates using jQuery
Smooth UI animations and transitions
📱 UI/UX Highlights
Clean dashboard layout
Animated circular timer
Responsive design
Interactive task panel
